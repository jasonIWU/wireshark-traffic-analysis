# Wireshark Traffic Analysis

## Overview
This project demonstrates network traffic analysis using Wireshark, focusing on key protocols such as DNS, TCP, TLS, and ICMP.

## Tools Used
- Wireshark
- macOS Terminal

## Key Skills Demonstrated
- Packet capture and analysis
- Protocol inspection (DNS, TCP, TLS, ICMP)
- Network troubleshooting
- Understanding encrypted traffic

- Report: Wireshark Traffic Analysis Report.pdf
- Screenshots: /screenshots/

## Summary
This project provided hands-on experience analyzing real network traffic and understanding how systems communicate securely across networks.
